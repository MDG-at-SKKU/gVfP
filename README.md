# genVESTAfromPOSCAR

## How to use

```
git clone https://github.com/MDG-at-SKKU/gVfP.git
cd gVfP
pip install -e .
gvfp
```

First time, `gVfP_config.ini` will be generated automatically.

Then set your input file name and name of result file name
